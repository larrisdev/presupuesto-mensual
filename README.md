# Presupuesto Mensual

Aplicación web desarrollada en JavaScript puro que permite gestionar ingresos y egresos, mostrando el estado financiero en tiempo real.

## Descripción

El sistema permite registrar transacciones de tipo ingreso y egreso, calcular automáticamente el balance disponible y mostrar el porcentaje de gastos.

## Funcionalidades

* Registro de ingresos y egresos
* Cálculo de totales y balance
* Porcentaje de gasto total
* Porcentaje individual en egresos
* Visualización por pestañas
* Eliminación de transacciones
* Validación de datos

* HTML5
* CSS3
* JavaScript
